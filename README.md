# AGRINHO-2026
Projeto agrinho
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente

Objetivo do Projeto
O objetivo deste site é conscientizar a comunidade escolar e a sociedade sobre a importância do equilíbrio entre a alta produtividade no agronegócio e a preservação do meio ambiente.

Justificativa
O tema "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente" proposto pelo Programa Agrinho exige uma reflexão. Este projeto web traduz conceitos fundamentais como o **Plano ABC (Agricultura de Baixo Carbono)**, o uso de energias renováveis (biodigestores e painéis solares) e a Integração Lavoura-Pecuária-Floresta (ILPF) em uma linguagem digital clara.
